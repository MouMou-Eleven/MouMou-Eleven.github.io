函数变换原理

对于原始函数 f(x)，我们可以通过以下变换得到新函数 g(x):

垂直拉伸/压缩

g(x) = a·f(x) 其中 a > 0

a > 1 时，图像在 y 方向拉伸

0 < a < 1 时，图像在 y 方向压缩

水平平移

g(x) = f(x - h)

h > 0 时，图像向右平移 h 个单位

h < 0 时，图像向左平移 |h| 个单位

垂直平移

g(x) = f(x) + k

k > 0 时，图像向上平移 k 个单位

k < 0 时，图像向下平移 |k| 个单位

水平拉伸/压缩

g(x) = f(b·x) 其中 b > 0

b > 1 时，图像在 x 方向压缩

0 < b < 1 时，图像在 x 方向拉伸

综合变换: g(x) = a·f(b·(x - h)) + k

实际应用示例

函数变换在数学建模、物理学、经济学等领域有广泛应用。例如：

  • 物理中的简谐运动可以用正弦函数的变换来描述
  • 经济学中的成本函数往往是二次函数的变换
  • 信号处理中，波形的变换可以通过函数变换来实现
